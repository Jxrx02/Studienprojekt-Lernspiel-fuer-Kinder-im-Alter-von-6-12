# Lernspiel-Prototyp – Tower-Defense mit Quiz-Integration

Dieses Repository enthält einen Prototypen eines Lernspiels, das klassische **Tower-Defense-Mechaniken** mit einer **lernorientierten Quiz-Integration** kombiniert. Ziel ist es, Lerninhalte spielerisch zu vermitteln, ohne den Spielfluss zu unterbrechen.

Das beigefügte Demo-Video zeigt den aktuellen Funktionsumfang und zentrale Gameplay-Elemente.

---

## Gezeigte Inhalte im Demo

### Allgemein
- **Hauptmenü**
- **Levelauswahl**
- **Ladebildschirm**
---

## Level 1 – Tower-Defense Grundlagen

In Level 1 liegt der Fokus auf klassischen Tower-Defense-Mechaniken:

- Platzierung von Türmen
- Reichweitenanzeige (Range Indicator)
- Zielauswahl / Target Selection
- Integration einer Lernfrage  
  - Erfolgreiche Beantwortung 

---

## Forschungsstation

Zwischen den Levels steht eine **Forschungsstation** zur Verfügung:

- Forschungspunkte werden durch das Abschließen von Levels verdient
- Forschungspunkte schalten **dauerhafte Turm-Upgrades** frei
- Freigeschaltete Upgrades wirken sich auf **nachfolgende Levels** aus

---

## Level 2 – Erweiterte Mechaniken

Level 2 erweitert das Gameplay um zusätzliche Systeme:

- Platzierung eines Helden
- Platzierung und Bewegung des Helden
- Aufwerten von Türmen
- Integration einer Lernfrage  
  - Fehlgeschlagene Beantwortung 

---

## Ziel des Prototyps

- Kombination von Lerninhalten und Gameplay ohne Unterbrechung des Spielflusses
- Modulare Erweiterbarkeit
- Grundlage für ein Lernspiel für Kinder im Alter von **6–12 Jahren**
